## 快速上手
```
$ npm run dev
```

## 打包
```
$ npm run build
$ npm run preview
```

## 计划实现
- 多语言
- 充值
- 数据报表
- 组合支付
- 积分功能
- 折扣功能
- 点餐功能

## 功能清单
- 支持会员和游客
- 支持SKU商品和普通商品
- 支持扫码枪扫码直接加购
- 支持扫会员码定位对应会员
- 支持挂单
- 支持整单取消
- 支持三种结算界面：会员支付、扫码支付、现金支付
- 支持订单
- 支持退款
- 支持小票打印
- 支持暗黑主题
- 支持全屏
- 支持会员的消费明细和资金记录

## 项目简介
- 基于arco design pro的vue3开发，纯前端的收银台解决方法，可自行整合后端。

## 使用
- 基于vue3和Arco Design写的项目，请参考各自文档
